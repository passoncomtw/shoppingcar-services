# ShoppingcarServices

## Services

### shoppingcar-service

```
$ nx serve shoppingcar-service --verbose
```
### shoppingcar-web
```
$ nx serve shoppingcar-web --verbose
```
### Shoppingcar-app

```
$ nx start shoppingcar-app --verbose
```
